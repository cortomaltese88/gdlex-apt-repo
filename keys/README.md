# Chiave pubblica del repository APT

In questa directory verra' pubblicata solo la chiave pubblica usata per la firma del repository APT GD LEX.

La chiave privata non deve mai essere committata nel repository, ne' copiata in file di lavoro permanenti.

Il file pubblico atteso, quando la firma sara' attivata, sara' ad esempio:

- `keys/gdlex-archive-keyring.asc`
