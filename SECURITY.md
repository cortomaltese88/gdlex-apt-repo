# Security Policy

- Non inserire segreti, token, credenziali o chiavi private in questo repository.
- Segnalare problemi di sicurezza tramite GitHub Issues oppure tramite i canali di contatto dello Studio GD LEX.
- Il repository APT e' firmato GPG; lato client la configurazione consigliata usa keyring locale e `signed-by=` invece di `trusted=yes`.
- Non caricare pacchetti `.deb` non verificati o provenienti da fonti non auditabili.
